# Boas Práticas de Senhas e Autenticação Multifator (MFA)

## 1. Boas Práticas de Senhas

Para garantir a segurança das suas contas, siga as seguintes práticas ao criar e gerenciar suas senhas:

*   **Use Frases Longas:** Prefira frases longas e complexas em vez de palavras curtas.
*   **Não Reutilize:** Nunca utilize a mesma senha em diferentes serviços.
*   **Evite Informações Pessoais:** Não inclua datas de nascimento, nomes ou sequências simples (como `123456`).
*   **Use Gerenciadores:** Utilize um gerenciador de senhas para armazenar todas as suas credenciais de forma segura.

## 2. O que é Autenticação Multifator (MFA)?

A Autenticação Multifator (MFA) é uma camada extra de segurança que exige mais de uma forma de verificação para confirmar sua identidade ao acessar uma conta. Isso torna muito mais difícil para invasores acessar suas contas, mesmo que consigam sua senha.

### Tipos de MFA

Existem diversos métodos de MFA disponíveis, incluindo:

*   Códigos enviados por SMS.
*   Códigos gerados por aplicativos (TOTP).
*   Chaves de segurança físicas.

## 3. Recomendação Corporativa: Microsoft Authenticator

Para reforçar a segurança das contas corporativas, recomendamos o uso do **Microsoft Authenticator** como método de Autenticação Multifator.

### Por que o Microsoft Authenticator?

O Microsoft Authenticator oferece uma forma segura e moderna de autenticação, aumentando significativamente a proteção das suas contas corporativas.

### Como Configurar o Microsoft Authenticator

Siga o passo a passo abaixo para vincular o aplicativo à sua conta corporativa:

1.  O usuário abre o aplicativo **Microsoft Authenticator**.
2.  O usuário aponta para o **QR Code** que aparece na tela do sistema corporativo.
3.  O usuário preenche o código gerado na tela do aplicativo.
