# Boas Práticas de Senhas

Senhas fortes são a primeira linha de defesa. Recomendações:

- Use **frases longas** em vez de palavras curtas.
- Não reutilize a mesma senha entre serviços.
- Ative a **autenticação multifator (MFA)** sempre que possível.
- Use um **gerenciador de senhas** para guardar credenciais únicas.

## O que evitar

Evite informações pessoais óbvias (datas de nascimento, nomes) e sequências como `123456`.
