# Introdução a Redes

Uma **rede de computadores** conecta dispositivos para que troquem dados. Os conceitos
fundamentais incluem:

- **Endereço IP**: identifica um dispositivo na rede (ex.: `192.168.0.10`).
- **Switch**: conecta dispositivos dentro de uma mesma rede local (LAN).
- **Roteador**: conecta redes diferentes e encaminha o tráfego entre elas.

## Como os dados trafegam

Os dados são divididos em **pacotes**, que viajam pela rede e são remontados no destino.
Cada pacote carrega o endereço de origem e de destino.
